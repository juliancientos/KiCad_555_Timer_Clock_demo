# 555 Timer Clock – 2‑Layer PCB Demonstration (KiCad)

## Overview
This project is a **beginner‑level demonstration of a 2‑layer PCB design** created in **KiCad**, using a simple **555 Timer Clock circuit** as the example.

The primary goal of this project is **to teach and practice basic 2‑layer PCB concepts**, not to create an optimized or production‑ready design. It is intended for students, hobbyists, and beginners learning PCB layout.

---

## Learning Objectives
This project demonstrates:
- Basic **2‑layer PCB layout** (Top & Bottom copper)
- Proper use of **ground planes**
- Via usage between layers
- Simple component placement and routing
- KiCad schematic → PCB workflow
